    10. **app/services/workspace.py** (per D-09, D-13 — env vars only):
        - Module-level `_registry: dict[str, str] = {}` mapping lowercase workspace name to API key
        - `WORKSPACE_ENV_PATTERN = re.compile(r'^WORKSPACE_([A-Z0-9_]+)_API_KEY$')`
        - `load_from_env()`: Reads all `WORKSPACE_*_API_KEY` env vars, populates `_registry`. Name transformation: `ENAVRA` -> `enavra`, `HEYREACH_CLIENT` -> `heyreach-client` (underscores become hyphens in display name).
        - `list_workspaces() -> list[dict]`: Returns `[{"name": "enavra", "key_preview": "...last4"}]`
        - `get_api_key(name: str) -> str | None`: Returns key for workspace name or None
        - `add_workspace(name: str, api_key: str) -> None`: Adds to `_registry` in-memory AND sets `os.environ[f"WORKSPACE_{name.upper().replace('-','_')}_API_KEY"] = api_key` so it's available for the current process
        - `remove_workspace(name: str) -> bool`: Removes from `_registry` and `os.environ`, returns True if existed
