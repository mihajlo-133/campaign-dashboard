    1. **app/routes/dashboard.py** — Open-access routes (per ADM-04):
       ```python
       from fastapi import APIRouter, Request
       from fastapi.responses import HTMLResponse
       from fastapi.templating import Jinja2Templates

       router = APIRouter()
       templates = Jinja2Templates(directory="app/templates")

       @router.get("/", response_class=HTMLResponse)
       async def dashboard(request: Request):
           return templates.TemplateResponse("dashboard.html", {"request": request})

       @router.get("/health")
       async def health():
           return {"status": "ok"}
       ```

    2. **app/routes/admin.py** — Password-protected routes (per ADM-01, ADM-02, ADM-03):
       - `router = APIRouter(prefix="/admin")`
       - `GET /admin/login` — Renders login.html (no auth required)
       - `POST /admin/login` — Accepts form data `password` field:
         - Calls `check_password(password)`
         - On success: creates session token via `create_session_token()`, sets `admin_session` cookie (httponly, path=/admin, samesite=lax), redirects to `/admin`
         - On failure: re-renders login.html with `error="Incorrect password. Try again."` (exact copy from UI-SPEC)
       - `GET /admin` — Protected by `require_admin` dependency:
         - Calls `list_workspaces()` to get current workspace list
         - Renders admin.html with `{"workspaces": workspaces}`
       - `POST /admin/workspaces` — Protected by `require_admin`:
         - Accepts form data: `workspace_name`, `platform`, `api_key`
         - Validates all 3 fields present (return 400 if missing)
         - Calls `add_workspace(workspace_name.strip(), api_key.strip())`
         - Returns HTML partial (for HTMX): updated workspace table row OR full redirect
         - On success: shows "Workspace added." toast
       - `POST /admin/workspaces/{name}/delete` — Protected by `require_admin`:
         - NOTE: Use POST with method name in path (not HTTP DELETE) for browser form compatibility
         - Calls `remove_workspace(name)`
         - Returns updated workspace list partial OR redirect
         - On success: shows "Workspace removed." toast
       - `GET /admin/logout`:
         - Clears `admin_session` cookie
         - Redirects to `/`

    3. **Update app/main.py** — Wire routers:
       - Add `from app.routes import admin, dashboard`
       - `app.include_router(dashboard.router)`
       - `app.include_router(admin.router)`
       - Ensure Jinja2Templates directory is set correctly (relative to where app runs: `app/templates`)
       - Add `from fastapi.staticfiles import StaticFiles` if needed for any static assets
